# Persephone

Persephone is a general-purpose simulation platform for local-first, plugin-driven simulation workflows.

The current implementation target is Version 1: a Python engine, a plugin SDK, typed experiment config validation, and a first runnable SIR epidemic simulation plugin.

See [Persephone.md](Persephone.md) for the architecture and [docs/tasks/v1_tasks.md](docs/tasks/v1_tasks.md) for the Version 1 implementation checklist.
